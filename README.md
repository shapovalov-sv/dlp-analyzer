# DLP Analyzer

Инструмент для анализа скриншотов рабочих станций и выявления признаков утечки данных. Помогает специалисту по экономической/информационной безопасности при первичном разборе DLP-инцидентов.

> ⚠️ Учебный pet-проект. Для тестов используются только синтетические скриншоты, не содержащие реальных корпоративных данных.

## Что это

Один файл — `dlp-analyzer-browser.html`. Скачайте его и откройте двойным кликом в браузере (Chrome/Edge). Установка не требуется: распознавание текста (OCR, русский + английский) и проверка правил выполняются прямо в браузере, файлы никуда не отправляются.

При первом запуске один раз подгружается движок распознавания (нужен интернет), дальше анализ работает локально.

## Возможности

- Загрузка скриншотов перетаскиванием, выбором файлов или целой папки
- Режим «Следить за папкой» (Chrome): новые скриншоты в папке распознаются автоматически, по завершении в папку сохраняется итоговый CSV
- Имя сотрудника определяется по имени папки; поддерживается структура `HS/<Сотрудник>/<дата>/скрины`
- Управление анализом: запуск / пауза / стоп с продолжением с того же места
- 29 категорий детекции (карты, паспорта, СНИЛС, пароли, коррупционная лексика, мессенджеры и др.) с включением/отключением каждой
- Предобработка изображений: тёмные темы осветляются, широкие (двухмониторные) кадры режутся пополам, мелкий текст увеличивается
- Фильтр OCR-шума от фотообоев по уверенности распознавания
- Дашборд по сотрудникам с риск-оценкой, разбивка по типам нарушений
- Таблица результатов с группировкой по сотрудникам (сворачиваемые группы), полным распознанным текстом, поиском и пагинацией
- Просмотр исходного скриншота из списка инцидентов
- Экспорт результатов в CSV (Excel-совместимый)

## Приватность

Вся обработка локальная: скриншоты, распознанный текст и результаты не покидают компьютер. Единственное сетевое обращение — одноразовая загрузка OCR-движка с CDN при первом запуске.

## Дисклеймер

Проект разрабатывался как личный тренировочный инструмент в рамках развития компетенций на стыке экономической безопасности и анализа данных. Не предназначен для production-использования и не связан с какой-либо корпоративной DLP-системой работодателя.
