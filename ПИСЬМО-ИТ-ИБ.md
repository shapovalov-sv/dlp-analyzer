**Кому:** ИТ-отдел, отдел информационной безопасности
**Тема:** Согласование инструмента для анализа скриншотов (DLP Screen Analyzer)

Добрый день, коллеги!

Прошу проверить на безопасность и согласовать к использованию в работе вспомогательный инструмент **DLP Screen Analyzer** для ускорения первичного разбора материалов DLP-расследований. Инструмент разработан мной, исходный код открыт и передаётся на проверку вместе с этим письмом.

**Что это.** Один HTML-файл (`dlp-analyzer-browser.html`), открывается в браузере (Chrome/Edge) двойным кликом. Установка не требуется — нет исполняемых файлов, служб, драйверов и изменений в системе.

**Что делает.** Распознаёт текст на скриншотах рабочих станций (OCR, рус+англ) и помечает кадры с признаками рисков: реквизиты карт и счетов, паспортные/персональные данные, пароли и токены, коррупционная лексика, признаки слива данных, личная почта, мессенджеры и др. Формирует отчёт (CSV/Excel), группирует по сотрудникам. Решение о квалификации инцидента принимает аналитик.

**Ключевое для проверки:**
- Вся обработка — **локальная, в браузере**. Скриншоты, распознанный текст и результаты **не покидают рабочую станцию**.
- Единственное сетевое обращение — **одноразовая загрузка OCR-движка** с публичного CDN при первом запуске; пользовательские данные не передаются. При необходимости движок можно разместить во внутренней сети (готов доработать).
- **Нет** серверной части, базы данных, телеметрии, автообновлений, учётных записей.
- Объём — один файл, читается за одну сессию; внешних зависимостей, кроме OCR-движка, нет.

**Чувствительность данных.** Инструмент обрабатывает скриншоты, которые могут содержать ПДн и конфиденциальную информацию. Предполагается использование на рабочей станции аналитика ИБ в рамках действующих регламентов; в отчётах чувствительные значения маскируются; новых мест хранения данных инструмент не создаёт.

**Прошу:**
1. проверить код на безопасность;
2. согласовать применение в работе;
3. при наличии требований для согласования (локальный OCR-движок, ограничения на сохранение отчётов и т. п.) — сообщить, внесу изменения.
